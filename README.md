# Nmax

Parser for a file. Searches for maximum values

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'nmax'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nmax

## Usage

cat <your_text_file> | nmax <value_of_count>
